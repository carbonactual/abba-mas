# ATLAS Public Proof After SEAL

## Purpose

ATLAS is the public visibility surface of Carbon Actual. It may show approved truth, public proof, public biography, public route, and public opportunity context.

ATLAS must not become a leak path for private memory, Vault content, regulated data, or true value.

## Publication law

```text
Memory stores masked reference.
ABBA resolves meaning.
SEAL approves publication.
Actual determines value visibility.
ATLAS shows only approved public truth.
```

## What ATLAS may show

- approved public summaries
- public proof references
- public source provenance
- public route status
- public opportunity context
- approved institutional records
- public-safe biography and achievements

## What ATLAS must not show

- raw memory
- private conversations
- raw identity files
- Vault records
- private keys
- actual value before authorization
- legal/financial/health details without SEAL-5 and lawful basis

## Provider route

ATLAS candidates:

- OriginTrail DKG
- Fabric Publish
- Obsidian Publish

## Activation path

1. Confirm masked event exists in `swarm_events`.
2. Confirm proof reference exists in `swarm_proof_refs`.
3. ABBA resolves what can become public.
4. SEAL approves publication.
5. Actual confirms value visibility.
6. ATLAS publishes public-safe record.

## Default status

Blocked until SEAL approval.
