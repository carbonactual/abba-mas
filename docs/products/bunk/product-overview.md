# BUNK — Product Overview

## Identity

BUNK is the property, place, shelter, land, development, ownership, investment, management and future-habitat branch of Carbon Actual.

## Core promise

BUNK helps a HAPI discover, verify, acquire, occupy, invest in, manage, maintain, trade and prove property across its full lifecycle.

## First complete journey

```text
Search Abuja 4-bedroom duplex
→ compare verified listings
→ inspect true total cost
→ review title and seller Proof
→ calculate BuyAbility
→ book inspection
→ make offer
→ negotiate
→ place deposit in escrow simulation
→ complete transfer evidence
→ add to portfolio
→ activate property management
```

## Experience standard

BUNK should feel calm, premium, spacious, map-aware and trustworthy. It should not feel like a crowded classifieds page. Proof, true cost, location and next action must always remain visible.
