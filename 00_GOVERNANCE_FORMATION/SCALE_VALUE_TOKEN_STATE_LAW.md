# Scale Value Token State Law

Token state is determined by Scale Value.

PULSE and VALUE are inputs.
Scale Value is the deciding foundation.

## Core correction

A token is not finally classified by raw PULSE alone.
A token is not finally classified by raw VALUE alone.
A token is not finally classified by a simple PULSE-to-VALUE ratio alone.

The final token state is determined by Scale Value.

## Meaning of Scale Value

Scale Value is the interpreted value after the ecosystem reads:

- PULSE strength
- VALUE returned
- level unlocked
- function activated
- proof attached
- cost carried
- risk carried
- time used
- effort used
- energy used
- network effect
- group effect
- future route
- recovery possibility
- field readiness
- SEAL permission

Scale Value decides what the token actually is.

## Token state

Scale Value can classify a token as:

- Asset Token
- Liability Token
- Equilibrium Token
- Mixed Token
- Pending Token
- Proof Token
- Warning Token
- Recovery Token
- Dormant Token
- Scale Token

## Role of PULSE

PULSE is what was released, emitted, carried, pledged, moved, spent, or sacrificed.

PULSE gives the token force and signal.

But PULSE alone does not decide final state.

## Role of VALUE

VALUE is what was created, returned, resolved, revealed, or made usable.

VALUE gives the token meaning and return.

But VALUE alone does not decide final state.

## Role of Scale

Scale reads the full context.

Scale determines level, weight, function, value, capacity, reach, consequence, and return.

Scale Value is the result of this reading.

The token state follows Scale Value.

## Corrected flow

Pledge releases PULSE.
Commitment creates VALUE.
PULSE is tokenized.
VALUE is read.
Scale reads both.
Scale Value is formed.
Token state is determined by Scale Value.
Base Value Token updates according to token state.

## Final law

Token state is determined by Scale Value.
PULSE is the emitted force.
VALUE is the created return.
Scale Value is the deciding classification.
The ecosystem does not classify tokens by raw signal or raw return alone.
The ecosystem classifies tokens by the scaled value of the whole formation.
