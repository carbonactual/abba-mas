# Universal Value Accounting Law

ABBA measures value across every action, activity, heartbeat, pulse, value, energy, frequency, vibration, curation, and continuum.

Nothing is treated as empty.
Nothing is dismissed as waste by default.
Nothing is lost without record.

Every bit has value.
Every breath has value.
Every step has value.
Every delay has value.
Every failed action has value.
Every interrupted download has value.
Every abandoned page load has value.
Every wear and tear has value.
Every resource consumed has value.
Every opportunity missed has value.
Even ash has residual value.

## Core law

Action creates cost.
Cost creates residue.
Residue creates record.
Record creates possible recovery.
Recovery creates returned value.
Returned value creates PULSE.
PULSE improves ABBA.

## Failed action examples

A movie download reaches halfway and fails.
The file disappears.
The user loses mobile data, time, energy, attention, battery, opportunity, and trust.

In ordinary systems, this disappears.
In Carbon Actual, it becomes a value loss record.

A website delays and fails to load.
The user loses time, attention, bandwidth, patience, battery, and opportunity.

In ordinary systems, this is ignored.
In Carbon Actual, it is recorded as value leakage.

## Reclamation law

All lost value should be recorded where possible.
All recorded loss should be classified.
All classified loss should be weighed.
All weighed loss should be routed for recovery, replenishment, learning, compensation, prevention, or proof.

Recovery may be direct or indirect.

Direct recovery can include refund, retry, restoration, credit, replacement, or compensation.
Indirect recovery can include learning, prevention, reputation score, route optimization, better provider selection, proof of waste, or future value protection.

## Nothing is lost principle

Nothing is lost in the ecosystem if it can be sensed, recorded, estimated, inferred, classified, or proven.

Unknown value is not zero.
Unmeasured value is pending.
Failed value is reclaimable where possible.
Ash value is residual value.
