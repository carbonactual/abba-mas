# Base Value Token Law

The Base Value Token is the currency of the ecosystem.

It is not only money.
It is the total value unit of Carbon Actual.

The Base Value Token represents the total value formed from all HAPI, products, organizations, companies, enterprises, groups, institutions, services, events, AI, commitments, pledges, scaled PULSE, and return PULSE in the ecosystem.

## Core meaning

Every ecosystem object can emit PULSE.
Every PULSE can be minted, tokenized, classified, scaled, and interpreted.
Every valid value can contribute to the Base Value Token.

The Base Value Token is the common value denominator of the ecosystem.

It allows different forms of value to be compared, routed, exchanged, scaled, stored, and sent.

## HAPI contribution

Each HAPI contributes value through presence, work, service, learning, skill, relationship, proof, commitment, reputation, network, and return PULSE.

HAPI value can be personal, social, professional, family-based, community-based, or ecosystem-wide.

## Product contribution

Products contribute value through use, service, adoption, reliability, proof, customer PULSE, integration, movement, income, saved time, solved problems, and future routes.

## Organization and enterprise contribution

Organizations, companies, institutions, and enterprises contribute value through services rendered, products delivered, staff performance, trust, compliance, network, revenue, social usefulness, proof, and ecosystem integration.

## Group contribution

Groups contribute value through collective PULSE.

A group may be classmates, workmates, friends, family, club, association, kinsmen group, department, project team, community, or event group.

Group AI reads emitted PULSE and forms bigger group value from combined commitments.

## Event contribution

Events contribute value through attendance, service, vendor movement, pledges, purchases, social closeness, family formation, community memory, proof, and future opportunities.

## Base Value Token formation

PULSE is released.
PULSE is minted.
PULSE is tokenized.
Token is classified.
Asset or liability state is read.
Scale is checked.
Valid value is measured.
Value contributes to Base Value Token.
Base Value Token updates the ecosystem value field.

## Asset and liability relation

The Base Value Token does not count value blindly.

Asset Tokens can increase Base Value.
Liability Tokens can reduce, warn, delay, or redirect Base Value.
Mixed Tokens may add or subtract depending on future return.
Pending Tokens wait for proof.
Recovery Tokens may restore value.
Scale Tokens determine how much weight a PULSE contributes.

## Currency of the ecosystem

The Base Value Token is the ecosystem currency because it converts different value forms into one readable unit.

Money, time, service, proof, trust, reputation, network, learning, commitment, protection, and opportunity can all be interpreted through Base Value.

## Final law

Base Value Token is the currency of the ecosystem.
It is the total value of all HAPI, products, organizations, groups, enterprises, institutions, services, events, AI, commitments, pledges, and scaled PULSE.
Every valid tokenized PULSE can contribute to Base Value.
Asset Tokens strengthen Base Value.
Liability Tokens reduce, warn, delay, or redirect Base Value.
Scale determines how much value is counted.
The ecosystem grows when its Base Value Token grows through real useful value.
