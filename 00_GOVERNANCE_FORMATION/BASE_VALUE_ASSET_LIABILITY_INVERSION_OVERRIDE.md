# Base Value Asset-Liability Inversion Override

This file corrects and overrides any earlier statement that says Asset Tokens strengthen Base Value or Liability Tokens reduce Base Value.

The correct law is:

Asset value reduces Base Value.
Liability value strengthens Base Value.

## Why

Carbon Actual economics is inverted.

A real-life asset can become ecosystem liability because it consumes wear, tear, time, effort, energy, cost, maintenance, attention, and burden.

A real-life liability can become ecosystem asset because it reveals pressure, need, gap, demand, service route, recovery route, opportunity, and movement.

## Base Value effect

Asset value reduces Base Value when the asset consumes more burden than it creates useful return.

Liability value strengthens Base Value when the liability creates demand, action, service, recovery, opportunity, proof, learning, or future route.

## Token effect

Asset Token means the ecosystem must account for consumption, burden, maintenance, depreciation, attention, and hidden cost.

Liability Token means the ecosystem can discover recoverable value, service value, demand value, pressure value, correction value, and opportunity value.
