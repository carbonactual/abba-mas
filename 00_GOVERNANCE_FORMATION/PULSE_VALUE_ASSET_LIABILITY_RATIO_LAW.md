# PULSE Value Asset-Liability Ratio Law

If PULSE is greater than VALUE, the token is an Asset Token.

If VALUE is greater than PULSE, the token is a Liability Token.

This is the corrected ratio law for Carbon Actual's inverted economics.

## Core meaning

PULSE represents what was released, emitted, carried, spent, moved, sacrificed, or pushed into the ecosystem.

VALUE represents what returned, resolved, created, or became usable after the PULSE was tokenized, scaled, and interpreted.

The relationship between PULSE and VALUE determines the asset or liability behavior of the token.

## Asset Token

A token becomes an Asset Token when:

PULSE > VALUE

This means the released PULSE is stronger than the immediate visible value.

In the ecosystem, this can behave as an asset because the excess PULSE carries stored potential, future route, scale, proof, network effect, or recoverable value.

The value may not have fully appeared yet, but the PULSE has force.

## Liability Token

A token becomes a Liability Token when:

VALUE > PULSE

This means the visible or claimed value is greater than the PULSE that supports it.

In the ecosystem, this can behave as a liability because the value may be overdrawn, unsupported, inflated, unearned, unstable, or carrying hidden burden.

## Equal state

When PULSE equals VALUE, the token is balanced.

PULSE = VALUE

This is an Equilibrium Token state.

The release and the return match.

## Inverted economics

Carbon Actual economics is inverted.

Ordinary economics may call something an asset because it has visible market value.

Carbon Actual checks whether the PULSE supporting that value is greater, equal, or weaker.

A high-value object with weak PULSE may become a liability.

A low-visible-value action with strong PULSE may become an asset.

## Examples

A person attends an event, serves, connects people, and creates future opportunity. The immediate value may look small, but the PULSE is large. This can be an Asset Token.

A person receives a big-looking benefit without enough PULSE, proof, work, commitment, or return route. The visible value is larger than the supporting PULSE. This can be a Liability Token.

## Final law

PULSE greater than VALUE creates Asset Token behavior.
VALUE greater than PULSE creates Liability Token behavior.
PULSE equal to VALUE creates Equilibrium Token behavior.
PULSE carries force.
VALUE carries revealed return.
The ratio between them determines token state under the inverted economics of Carbon Actual.
