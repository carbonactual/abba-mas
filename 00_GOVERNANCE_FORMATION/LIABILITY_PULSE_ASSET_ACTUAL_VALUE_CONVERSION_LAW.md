# Liability PULSE and Asset Actual Value Conversion Law

Every liability is released as PULSE within the ecosystem.

An asset is not automatically released as PULSE.

An asset must be converted to PULSE through Actual Value.

## Core meaning

A liability naturally produces pressure.

Pressure releases PULSE.

This is why every liability can enter the ecosystem as PULSE: debt, burden, need, damage, risk, failure, obligation, delay, pain, shortage, waste, or unresolved duty all release signal.

An asset is different.

An asset may be present, owned, stored, named, priced, or visible, but it does not automatically move in the ecosystem.

For an asset to become PULSE, Actual Value must be read from it.

## Liability as PULSE

A liability releases PULSE because it creates motion, demand, warning, pressure, urgency, obligation, or need for resolution.

Examples:

- debt releases PULSE
- failed pledge releases PULSE
- broken trust releases PULSE
- unpaid work releases PULSE
- wasted time releases PULSE
- burden releases PULSE
- risk releases PULSE
- damage releases PULSE
- unmet need releases PULSE

The ecosystem reads the liability PULSE and determines whether it becomes warning, recovery, asset potential, redress, containment, or future route.

## Asset conversion

An asset must be converted to PULSE through Actual Value.

Actual Value means the asset has been read, proven, made useful, routed, activated, or connected to real return.

Until Actual Value appears, the asset may remain dormant, decorative, idle, inflated, locked, or even a hidden liability.

## Actual Value conversion path

Asset is identified.
Actual Value is checked.
Usefulness is proven.
Scale is read.
Field checks readiness.
Asset is converted into PULSE.
PULSE is tokenized.
Token state is determined by Scale Value.
Base Value Token updates.

## Inverted economics

A real-life asset may be ecosystem liability if it does not convert into Actual Value.

A real-life liability may become ecosystem asset because it releases PULSE that can be read, recovered, redirected, or transformed.

## Final law

Liability releases PULSE.
Asset requires Actual Value before it can become PULSE.
Liability enters through pressure.
Asset enters through conversion.
Actual Value is the bridge that converts asset into usable PULSE.
Scale Value determines the final token state.
