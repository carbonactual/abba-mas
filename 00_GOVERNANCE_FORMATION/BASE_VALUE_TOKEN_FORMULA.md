# Base Value Token Formula

The Base Value Token is the ecosystem currency.

It is the total readable value field of Carbon Actual after PULSE has been tokenized, classified, scaled, and interpreted.

## Core formula

Base Value Token is affected by all valid tokenized PULSE across HAPI, AI, groups, entities, products, services, organizations, institutions, events, vendors, commitments, pledges, proof, and return PULSE.

## Inverted economics rule

Asset value reduces Base Value.
Liability value strengthens Base Value.

## Base Value movement

Base Value increases when liability pressure becomes useful movement, service, recovery, correction, demand, opportunity, proof, learning, or future route.

Base Value reduces when asset pressure consumes more wear, tear, time, effort, energy, cost, burden, maintenance, attention, or risk than it returns.

## Formula language

For each token:

Scaled Liability Value strengthens Base Value.
Scaled Asset Burden reduces Base Value.
Pending Value waits for proof.
Recovery Value restores Base Value.
Warning Value protects Base Value by preventing repeated loss.
Proof Value stabilizes Base Value.

## Practical expression

Base Value Token = total scaled liability value + total recovery value + total proof value + total useful return value - total scaled asset burden - unresolved damaging burden.

## Scale factor

Scale determines how much a PULSE affects Base Value.

Scale reads weight, level, function, reach, capacity, proof, risk, group effect, network effect, service effect, timing, and return strength.

## Return PULSE

Return PULSE can update the formula.

A token may move from pending to liability value, liability value to recovery value, asset burden to warning, or mixed token to stronger Base Value depending on what actually happens.

## Final law

Base Value Token is not a simple money balance.
It is the living value field of the ecosystem.
Liability value strengthens it.
Asset value reduces it.
Scale determines the size of the effect.
Proof stabilizes it.
Return PULSE updates it.
