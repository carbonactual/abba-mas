# Token State Machine and Value Ledger Law

This law defines how PULSE becomes token, how token becomes readable value, and how the ledger records ecosystem movement.

## Core rule

PULSE is not value by default.

PULSE must be minted, sorted, tokenized, classified, scaled, checked, and interpreted before value is revealed, sent, contained, or recovered.

## Token state machine

The token state machine follows this route:

1. Signal enters
2. ABBA reads
3. PULSE is formed
4. PULSE is minted
5. PULSE enters Terminal
6. Terminal sorts
7. PULSE is tokenized
8. Token is classified
9. Scale is read
10. Field is checked
11. SEAL is checked where required
12. Value is interpreted
13. Base Value effect is applied
14. Ledger records state
15. Actual reveals permitted value
16. Return PULSE updates the token

## Asset and liability classification

Carbon Actual economics is inverted.

Asset value reduces Base Value.
Liability value strengthens Base Value.

Asset Token records consumption pressure, maintenance, burden, wear, tear, time, effort, cost, and hidden load.

Liability Token records pressure, need, gap, demand, recovery route, service route, opportunity, correction, and movement.

## Ledger purpose

The Value Ledger records:

- who emitted PULSE
- what emitted PULSE
- why the PULSE was emitted
- what token was formed
- asset or liability state
- scale
- proof
- field readiness
- SEAL status where needed
- Base Value effect
- Actual reveal status
- return PULSE
- final or pending state

## Ledger routing

Root stores deep identity, origin, lineage, obligations, and source truth.
Index stores organized searchable intelligence.
Vault stores sensitive proof, rights, contracts, credentials, and protected evidence.
Actual reveals qualified value.
Terminal sorts incoming PULSE before tokenization.
I/O moves value, PULSE, proof, and return.

## Final law

PULSE becomes value only through the token state machine.
The ledger records the state, proof, scale, route, and Base Value effect.
Asset value reduces Base Value.
Liability value strengthens Base Value.
Actual reveals only the value that is permitted, scaled, proven, and routed.
